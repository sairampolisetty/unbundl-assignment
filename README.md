--- Clove Dental Landing Page ---

A fully responsive landing page for Clove Dental, built using HTML, CSS, and JavaScript (no frameworks).
This project visually matches the provided design screenshots and is ready for deployment on GitHub Pages.

### Features
Responsive design (mobile, tablet, desktop)
Semantic HTML5 structure
Modern CSS (Flexbox, Grid, media queries)
FontAwesome icons and Google Fonts
Interactive navbar (mobile toggle)
FAQ accordion
Smooth scroll navigation
Placeholder images (easily replaceable)
Clean, maintainable code
Getting Started

***Clone the repository:***
Apply to index.html
Run
Open the landing page:
Simply open index.html in your browser.
